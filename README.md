# 🛒 E-commerce Data Analysis Project
Ecommerce_Dashboard.pdf
## 📌 Overview

This project focuses on analyzing an e-commerce dataset using SQL and Excel to generate business insights such as sales performance, customer behavior, and product trends.

---

## 🛠 Tools Used

* SQL (Data Cleaning & Analysis)
* Excel (Charts & Dashboard Visualization)

---

## 📂 Dataset

The dataset includes the following columns:

* User ID
* Product ID
* Category
* Price
* Discount
* Final Price
* Payment Method
* Purchase Date

---

## 🔍 Steps Performed

### 1. Data Import

* Imported CSV dataset into SQL database

### 2. Data Cleaning

* Handled NULL values
* Ensured correct calculation of final price

### 3. Data Analysis using SQL

Performed multiple analyses including:

* Total Revenue
* Total Orders
* Average Order Value
* Category-wise Revenue
* Top Products
* Customer Spending
* Payment Method Analysis
* Monthly Sales Trends

(All queries are available in 'ecommerce_dataset_analysis.sql' file)

---

## 📊 Dashboard

* Created an Excel dashboard using SQL outputs
* Includes:

  * Revenue by Category
  * Monthly Sales Trend
  * Payment Method Distribution

---

## 📁 Project Files

* `ecommerce_dataset_updated.csv` → Raw dataset
* `ecommerce_dataset_analysis.sql` → All SQL queries used
* `ecommerce_dataset.xlsx` → Excel dashboard
* `Ecommerce_Dashboard.pdf` → Final dashboard presentation

---

## 💡 Key Insights

* Certain product categories contribute the highest revenue
* Customer spending varies significantly across users
* Payment method usage shows clear preference trends
* Sales vary across different months

---

## 🚀 Conclusion

This project demonstrates how SQL and Excel can be used together to clean, analyze, and visualize e-commerce data to derive meaningful business insights.
